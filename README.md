# Mini Renderer
A simple CPU renderer.

## Minimum Requirements
  
Operating System: **Windows** / **Linux**  
Cmake: version 3.6  
C++ compiler with C++14 support  

## How to build
  
For Windows  
```  
cmake -S . -B ./bin
```  
  
For Linux  
```  
cmake -S . -B ./bin -DPLATFORM_WINDOWS=OFF
```
